# Titanic Survival Analysis

## 📌 Objective
Analyze Titanic passenger data to understand factors affecting survival.

## 🛠 Tools Used
- Excel
- Pivot Tables
- Data Visualization

## 📊 Key Insights
- Female passengers had higher survival rates than males
- First-class passengers had better survival chances
- Children had higher survival probability
